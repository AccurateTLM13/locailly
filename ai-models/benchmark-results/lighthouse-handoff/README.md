# Lighthouse Handoff benchmark results

Side-by-side validation summaries for console L2 runs.

Generated by:

```powershell
node scripts/benchmark-lighthouse-handoff.js --mode l2_ollama_memory --url https://example.com/
```

Each run uses the same URL/mode and only changes the Ollama model via `--model`.

Compare:

| Metric | Where to look |
| --- | --- |
| Runtime | `result.durationMs` in summary |
| Schema | `result.schemaValid` |
| Memory Bridge | `result.filesUsed`, `result.warnings` |
| Metric preservation | `evidence.metricPreservation` |
| Model used | top-level `model` and `modelSlug` |

Commit result JSON files here when a benchmark pass is worth keeping.
