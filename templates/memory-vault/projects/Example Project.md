# Example Project

Sample project page for the flat starter template.

## Current state

Memory Bridge v0 provides read-only context packs and proposal-only writeback.

## Decisions

- Use flat `projects/` and `topics/` in the public starter template.
- Private Second Brain vaults can enable `wiki/` paths via config.

## Constraints

- No private content in the Locaily public repo.
- Context packs return summaries and excerpts, not full file dumps.
- Writeback is proposal-only until human review.
